HTML5 & CSS Normalize

HTML5 is becoming more popular since its release few years ago to many web developers because of its new features . These new features include the introduction of new elements such as <video>, <audio> and <canvas> which makes it easier to manage multimedias. Cross-platform compatibility which makes web apps run to all types of devices and in all kinds of mobile operating systems is the most interesting aspect of HTML5. On the other hand, normalize.css provides default styling of HTML elements to all types of browsers making sure that all web apps will appear the same as each browser have different default settings.
