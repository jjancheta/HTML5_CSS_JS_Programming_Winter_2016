HTML5

HTML5 new form attributes

HTML5 introduces new input attributes and values. This includes "autofucus", "placeholder", "required" and "list" as new attributes. New input type values such as "email" and "url" are also added which makes it easier to validated inputs of this format.


How to set autofocus


<input type="text" name="entry01" id="entry01" autofocus="autofocus">

How to set placeholder


<input type="text" name="entry02" id="entry02" placeholder="Passport Number">

How to set required for an entry


<input type="text" name="entry03" id="entry03" required="required" placeholder="Required ID"/>

How to create a list/datalist


<input list="entry04" name="entry04" placeholder="Marital Status">
<datalist id="entry04">
<option value="Single">Single</option>
<option value="Married">Married</option>
<option value="Widow/Widower">Widow/Widower</option>
<option value="Divorced">Divorced</option>
<option value="Complicated">Complicated</option>
</datalist>

How to ensure a valid email address format


<input type="email" name="entry05" id="entry05" placeholder="Email">

How to ensure a valid url format


<input type="url" name="entry06" id="entry06" placeholder="URL">

New HTML5 Attributes & Values

  1. autofocus="autofocus"
  2. placeholder="Passport#"
  3. list="status"
    datalist id="status"
    option value="Single"
  4. type="email"
  5. type="url"
